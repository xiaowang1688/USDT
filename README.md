## 1. 简介

本教程将详细介绍如何在TRON区块链上编写、编译和部署一个简单的TRC20代币智能合约。我们将使用TronBox工具进行合约的编写和部署。

## 2. 前提条件

在开始之前，请确保已安装以下工具和环境：

- Node.js
- TronBox
- Solidity编译器
- TRON钱包和测试网（Shasta）账户

您可以通过以下命令安装TronBox：

```bash
npm install -g tronbox
